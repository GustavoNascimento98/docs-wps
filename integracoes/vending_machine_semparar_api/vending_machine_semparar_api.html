    Integração Vending Machine / Sem Parar API

    Informações gerais

    Este documento contém informações sobre a API de integração dos dados da Vending Machine com o Sem Parar, permitindo o compartilhamento de todos os registros coletados na operação Vending Machine ( Entrega de Cortesias / TAG Free ) - Pague Fácil.

    Versão mínima Parkingplus

    Versão mínima do sistema: 7.376.1

    Versão mínima Enterprise

    Versão mínima do sistema: 1.135.0

    Requisitos

    A operação Vending Machine (Venda de produtos / TAGs) deve estar ativa na garagem, se necessário consulta o manual Vending Machine, para ativação da msm.

    Ativação API

    Para ativar a API é necessário adicionar as seguintes chaves na propreties do Enterprise.

    1 - Ativação API 

    integracaovendingmachine.ativar=true

    2 - Ativação Schedures de envio dos dados

    scheduler.integracaovendingmachine.sincronizacao.ativar=true

    Obs : As chaves abaixo só são necessárias caso deseje alterar os valores default.

    3 - Cron de envio dos dados - Default: Cada 2 minutos

    scheduler.integracaovendingmachine.sincronizacao=0 */2 * * * ?

    4 - Quantidade de registros a serem processados a cada integração - Default: 50 registros

    integracaovendingmachine.registros.limit=50

    5 - Nome do Integrador - Default: SEM_PARAR

    integracaovendingmachine.integrador.nome=SEM_PARAR

    6 - Chave da Operação Sem Parar - Default: MKUtDHB7yJjGWqBvBDBywwim4l5wwwQxqU=

    integracaovendingmachine.operacao=MKUtDHB7yJjGWqBvBDBywwim4l5wwwQxqU=

    7 - URL API - Default: http://webproc.intergrall.com.br/webservices/intergrall-api/semparar/callback/incluircallback

    integracaovendingmachine.ws.url=http://webproc.intergrall.com.br/webservices/intergrall-api/semparar/callback/incluircallback

    8 - Timeout conexão - Default: 20 Segundos

    integracaovendingmachine.ws.timeout=20

    Campo Descrição Obrigatório
    Nome Nome do cliente Sim
    DDDTelefone DDD Telefone do cliente Sim
    Telefone Telefone do cliente Sim
    Email E-mail do cliente Não
    DDDCelular DDD Celular do cliente Não
    Celular Celular do cliente Não
    Observacao Texto de detalhes ou informações adicional Não
    IdInclusor Identificador da estação Sim
    Operacao Chave única fornecida pelo Sem Parar Sim
    OrigemLead Identificador único da garagem Não
    Cep CEP do cliente Não
    Endereco Endereço do cliente Não
    Complemento Complemento endereço do cliente Não
    Bairro Bairro do cliente Não
    Cidade Cidade do cliente Não
    Estado Estado do cliente Não
    Sexo Sexo do cliente Não
    DataNascimento Data de Nascimento do cliente Não
    EstadoCivil Estado civil do cliente Não
    Rg RG do cliente Não
    Cpf_Cnpj CPF ou CNPJ do cliente Não
    Qtd_veiculos Quantidade de veículos Não
    Empresa Empresa integradora Não
    Produto Descrição do produto Não
